Так як у нас не співпадають айдішки юзера, профайла і аккаунта, потрібно зробити щоб при запиті на 
api/user/<int:pk>/account 
повертався аккаунт саме того юзера якого ми вказуємо в <int:pk>, теж саме і з профайлом зробити.

А вивід профайлів і аккаунтів я думаю доречніше зробити звертаючись до них одразу по типу:
api/account/list
api/account/<int:pk>

бо api/user/<int:pk>/account і api/account/<int:pk> це будуть різні по факту аккаунти

якщо що, я в settings.py замінив дані для бд, тому створиш .env файл вже для себе)

і сваггер додав, можеш через браузер дивитися тепер як ти хотів.

pip install requirements.txt
